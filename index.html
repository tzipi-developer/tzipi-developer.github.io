Системное администрирование с InfoBit

Улучшайте свои навыки с нами

Это InfoBit, проект разрабатываемый в рамках курсовой работы в учебном учреждении "Саранский политехнический техникум". Проект включает в себя разработку ЭОР на базе сайта, реализуется проект с помощью созданной текущей версией сайта.

Реализован ЭОР в виде сайта с тематической информацией по специальности "Системное администрирование", содержит в себе теоретический материал по темам, практические работы, презентации, видеокурсы и другую информацию.

Данный проект переходная стадий в дипломную работу по данной теме, которая будет реализована в форме мобильного приложения, нести в себе больше информативности и интерактивности.
